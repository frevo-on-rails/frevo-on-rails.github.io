Guia de Organização

    Lugar

    O lugar onde o evento acontece deve ser um espaço aberto para acomodar os participantes e ter um espaço para projeção da apresentação e movimentação dos palestrantes.

    Criar um evento no Sympla ou um formulário para registrar os participantes deve dar uma boa ideia de quantas pessoas vão atender ao evento.

    Formato

    Não temos um formato bem definido para todos os encontros, cada organizador pode ficar a vontade para escolher o formato. Alguns formatos mais utilizados são palestras de 20 minutos e discussões abertas entre participantes.

    Palestrantes

    Encontrar palestrantes pode ser demorado mas não é muito difícil. Basta mandar um email para o grupo ou divulgar que o evento está procurando palestrantes.

    É interessante tentar balancear o conteúdo das palestras para todos os níveis de público. Assim iniciantes podem se sentir motivados e os mais experientes também ficarão interessados.

    Lanches

    Essa é uma parte muito importante do evento! Geralmente separamos uma faixa um pouco maior do que uma palestra para que os participantes possam conversar e socializar.

    Divulgação

    O primeiro passo para divulgar o encontro é criar um novo post. Para isso faça um fork do repositório original e crie um novo post seguindo o formato de posts anteriores.

    Uma boa ideia para começar a divulgação é enviar um email para o grupo e pedir que os membros compartilhem tweets, posts no facebook, etc. Além disso compartilhe o evento com outros grupos para chamar novos membros.
